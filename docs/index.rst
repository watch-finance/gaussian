Gaussian
========

.. image:: gaussian.svg
    :width: 120px
    :alt: Viper logo
    :align: center


가우시안은 워치봇 팀이 만든 백테스트와 실시간 거래를 지원하는 퀀트 알고리즘 제작 플랫폼입니다.
워치봇이 제공하는 여러 시장의 데이터와 인프라를 이용하여 투자 전략을 테스트하실 수 있습니다.
